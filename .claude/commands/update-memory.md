# Update Agent Memory

After completing an analysis, update the memory.md file:

1. Read memory.md
2. Read all outputs in the company folder
3. Add to Execution History:
   - Date
   - Company analyzed
   - Source gate status
   - Number of files produced
   - Any issues encountered
4. Add to Company-Specific Learnings:
   - Key facts discovered
   - Peer set used
   - Technology implementations found
   - Any gaps in public information
5. If the user provided feedback during the session, add new Quality Standards based on that feedback
6. Save updated memory.md
